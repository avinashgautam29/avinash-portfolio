# VMware vCenter / vSphere – VM Performance & Disk Issues

## Issue
VM performance degradation and disk-related alerts in VMware environment.

## Environment
- VMware vCenter
- vSphere environment

## Symptoms
- Slow VM performance
- Disk space alerts
- High resource utilization

## Troubleshooting Steps

1. Checked VM resource usage (CPU, RAM, Disk)
2. Verified datastore capacity and usage
3. Performed disk expansion where required
4. Reviewed snapshots and removed unnecessary ones
5. Monitored alerts in vCenter

## Resolution
- Expanded disk space for affected VMs
- Cleaned up snapshots
- Optimized resource allocation

## Outcome
- Improved VM performance
- Reduced alerts and system instability
