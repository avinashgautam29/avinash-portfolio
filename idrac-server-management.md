# Dell iDRAC – Remote Server Management

## Tasks Performed
- Remote server monitoring
- Power cycling servers
- Hardware-level troubleshooting

## Troubleshooting Steps

1. Accessed server via iDRAC console
2. Checked hardware logs and alerts
3. Performed remote reboot/power cycle
4. Diagnosed hardware-related issues

## Outcome
- Faster resolution of server issues
- Reduced need for physical intervention
