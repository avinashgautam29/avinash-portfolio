# Proxmox VE – VM Management & Troubleshooting

## Tasks Performed
- VM creation and configuration
- Resource allocation (CPU, RAM, Storage)
- Basic troubleshooting of VM issues

## Common Issues
- VM not starting
- Resource constraints
- Storage-related errors

## Troubleshooting Steps

1. Checked VM status via Proxmox dashboard
2. Verified resource allocation
3. Reviewed storage availability
4. Restarted VM and services if required
5. Checked logs for errors

## Outcome
- Restored VM functionality
- Ensured stable performance of virtual machines
